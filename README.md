🌴 Disney World & Universal Orlando Trip Planner
A fully custom, hand‑built vacation planning web app
Created by Brian — my first ever attempt at programming

📌 Overview
This project is a custom-built, interactive trip‑planning web application designed to organize an 8‑day family vacation 

The app includes:
Day‑by‑day itineraries
Real Google Maps routing links
Real dining menu links
Flight plans with seat assignments
Resort details and Club Level amenities
Tools & Extras pages
A clean dark‑mode UI
Mobile‑friendly layout
Interactive cards, buttons, and navigation
Everything is structured so the entire trip can be viewed, edited, and navigated easily from any device.

🚀 Deployment
This project is designed to run on GitHub Pages.
Once pushed to GitHub:
Go to Settings → Pages
Select the main branch
Set root folder
Save
Your site will deploy automatically
